# Version etudiant vs recherche

## Version etudiant

But:

- comprendre;
- apprendre;
- formaliser;
- faire des exercices;
- produire une note courte.

Niveau:

- licence avancee;
- master debutant;
- tuteur GPT pedagogique.

Livrables:

- fiches;
- exercices;
- oral;
- mini-protocole.

## Version recherche

But:

- produire un probleme original;
- tester un claim;
- construire un protocole;
- preparer replication;
- publier une note academique prudente.

Niveau:

- master recherche;
- doctorat;
- enseignant;
- revue critique.

Livrables:

- article court;
- protocole preregistre;
- annexe mathematique;
- registre d'incertitude;
- plan de validation externe.

## Passage de l'un a l'autre

Le passage se fait quand l'etudiant peut:

- citer les sources;
- definir les objets;
- prouver localement;
- refuser les surclaims;
- construire un baseline;
- ecrire une condition d'echec.
