# Presentation publique etudiants

## Description

Ce pack est une entree pedagogique qui utilise le corpus MDL Ynor actuel comme support d'entrainement. Il vise les etudiants qui veulent apprendre a exploiter une base documentaire complexe avec methode, prudence et rigueur.

La finalite n'est pas de servir une base documentaire. La finalite est d'aider les etudiants a exceller en utilisant les sources comme support d'entrainement, de recherche, de formalisation et de critique.

## Ce qui le distingue

- architecture dediee aux etudiants;
- progression par niveau;
- noyau mathematique explicite;
- protocole de validation scientifique;
- exercices corriges;
- templates de recherche;
- prompts GPT controles;
- rubriques d'evaluation.

## Usage en cours

Un enseignant peut utiliser le pack pour:

- organiser un seminaire;
- donner un devoir de formalisation;
- evaluer un claim;
- enseigner la difference entre preuve locale et validation externe;
- entrainer les etudiants a produire des conclusions prudentes.

## Usage individuel

Un etudiant peut utiliser le pack pour:

- comprendre les notions utiles a leur progression;
- preparer une note;
- faire des exercices;
- construire un protocole;
- preparer un oral;
- utiliser un GPT tuteur sans perdre le controle critique.
- transformer les sources en competences academiques concretes.

## Formulation prudente

Le pack est un support pedagogique et methodologique. Il ne certifie pas les claims internes et ne remplace pas une validation independante.
