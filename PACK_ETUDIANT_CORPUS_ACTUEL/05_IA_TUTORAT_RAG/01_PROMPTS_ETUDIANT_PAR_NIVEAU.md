# Prompts etudiant par niveau

## Prompt lecture rapide

```text
Explique-moi cette notion utile a mon apprentissage en trois niveaux: intuition, definition, limite. Si elle vient des sources internes, cite le type de source utilise: CANON, RAG_SUPPORT ou AUDIT. Termine par une question pour verifier ma comprehension.
```

## Prompt simplification utile

```text
Utilise les sources complexes en arriere-plan, mais ne me montre que ce qui m'aide a progresser. Donne une explication simple, une methode applicable, une limite a retenir et un exercice court. Ne detaille l'architecture documentaire que si je te le demande.
```

## Prompt fiche academique

```text
Transforme cette notion en fiche academique avec: source, definition, domaine, exemple, contre-exemple, statut epistemique, lien avec alpha/beta/kappa/mu si pertinent, et point a verifier.
```

## Prompt mathematique

```text
Formalise cette idee proprement. Donne les objets, le domaine, les definitions, les hypotheses, une proposition, une preuve ou esquisse, puis les limites. Si ce n'est pas prouvable, dis-le.
```

## Prompt scientifique

```text
Transforme ce claim en protocole testable avec observable, methode, snapshot/fenetre, baseline, condition d'echec, incertitude, couverture et statut attendu.
```

## Prompt recherche web

```text
A chaque reponse, utilise Internet ou tente une verification web. Priorise articles scientifiques, preprints identifies, documentation officielle et institutions. Compare avec les sources internes utiles, cite chaque source avec lien, date et statut, puis distingue convergence, contradiction, incertitude et prochaine verification. Si aucune source pertinente n'est trouvee ou si l'outil web est indisponible, indique-le dans une section Recherche web.
```

## Prompt critique de claim

```text
Evalue ce claim avec la grille academique: source, domaine, observable, methode, baseline, incertitude, validation interne/externe, condition d'echec. Donne un verdict prudent et une formulation corrigee.
```

## Prompt revision de memoire

```text
Relis cette note comme un directeur de memoire exigeant. Signale les definitions floues, preuves manquantes, surclaims, baselines absents, incertitudes cachees, confusions entre support et canon. Propose une version corrigee du plan.
```

## Prompt oral

```text
Prepare-moi a soutenir cette idee en 12 minutes. Donne le plan, les 5 questions difficiles probables, les reponses prudentes et les limites a assumer.
```

## Prompt niveau debutant

```text
Explique sans jargon inutile, mais garde la rigueur. Donne une analogie courte, puis la definition exacte, puis un exercice simple corrige.
```

## Prompt niveau master

```text
Reponds au niveau master: definitions precises, hypotheses, preuve locale si possible, discussion des observables, protocole empirique, baseline, incertitude et objections.
```

## Prompt niveau doctorat

```text
Traite cette question comme un objet de recherche: problematisation, lacune, formalisation, hypothese falsifiable, protocole, risques de circularite, validation externe et plan de replication.
```
