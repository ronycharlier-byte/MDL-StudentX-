# MDL StudentX

Pack academique pour creer un GPT custom de tutorat universitaire rigoureux, centre sur la reussite des etudiants.

MDL StudentX fournit une base de connaissances, une grille de lecture et des garde-fous academiques pour aider un GPT custom a produire des reponses pedagogiques, scientifiques, prudentes et exploitables par des etudiants de niveau licence avancee, master ou doctorat.

## Description courte pour GitHub

Pack academique pour GPT custom: tutorat etudiant, methodologie universitaire, recherche web, hypotheses, protocoles, soutenance et prudence epistemique.

## Principe

Le corpus sert l'etudiant. L'etudiant ne sert pas le corpus.

## Pour qui ?

- etudiants de licence avancee, master et doctorat;
- enseignants, tuteurs, formateurs et responsables pedagogiques;
- createurs de GPT custom academiques;
- organismes voulant deployer un assistant pedagogique prudent.

## Contenu

- `PACK_ETUDIANT_CORPUS_ACTUEL/`: dossier de connaissance a joindre au GPT custom.
- `GPT_UPLOAD_PACK_ETUDIANT_ACADEMIQUE_2026-05-05.zip`: archive prete a televerser dans le GPT custom.
- `LICENSE.md`: licence non commerciale academique.

## Usage GPT custom

1. Coller `PACK_ETUDIANT_CORPUS_ACTUEL/07_GPT_UPLOAD/GRILLE_LECTURE_GPT_CUSTOM.md` dans le champ Instructions du GPT.
2. Joindre le zip comme base de connaissances.
3. Activer la recherche web si disponible.

## Ce que le GPT doit produire

- tutorat academique rigoureux;
- aide aux memoires, protocoles, hypotheses et soutenances;
- prudence scientifique, mathematique et epistemique;
- recherche web propre avec URL brutes;
- exercices pour verifier la comprehension.

## Points forts

- instructions GPT custom pretes a coller;
- zip pret a televerser;
- architecture pedagogique multi-domaines;
- modules psychologie, methodologie, validation scientifique, mathematiques et evaluation;
- format de reponse avec sources internes courtes, recherche web propre, statut epistemique, limites et exercice;
- garde-fous contre hallucination, surpromesse et fausses validations externes.
