# Release v1.0

MDL StudentX v1.0 is the first public release of the student academic GPT pack.

## Purpose

Help students produce stronger academic work by giving a GPT custom:

- a clear tutoring identity;
- a disciplined response format;
- web research requirements;
- epistemic safeguards;
- methodology and validation templates;
- exercises that verify student understanding.

## Included

- full `PACK_ETUDIANT_CORPUS_ACTUEL` folder;
- ready-to-upload GPT knowledge zip;
- GPT custom instructions grid;
- license and commercial contact files.
